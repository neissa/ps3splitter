# How to #

Lancer le programme, sélectionnez le dossier qui contient le jeu, puis cliquer sur "Lancer le découpage", le programme trouve les fichiers de + de 4Go et les découpe.

Puis transférez les avec Comgenie’s Awesome Filemanager sur le disque interne de votre PS3

**ATTENTION : prévoyez de l'espace disque pour le traitement et faite une sauvegarde de votre jeu si possible avant le découpage.**
